# Bahasa Indonesia Checker

Aplikasi web sederhana untuk membantu memeriksa kata dalam teks Bahasa Indonesia.
Pemeriksaan dilakukan dengan mencocokkan kata terhadap daftar kata baku.
Kata berimbuhan akan diberi keterangan karena belum didukung analisis imbuhan.
Proyek ini dibuat sebagai media pembelajaran Bahasa Indonesia berbasis web.
Teknologi yang digunakan: HTML, CSS, JavaScript, dan JSON.
